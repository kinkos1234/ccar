- 언어에 따라 일부 문장 길이 조정 (예: 스페인어는 요약 길이 짧게 조정)

### 2.3 Date Score 통합 분석 시스템 (2025-07-02)

#### 2.3.1 Date Score 분석 개념
- **정의**: CONTINUOUS 이벤트의 시간 관리 효율성을 AI가 종합 분석
- **적용 범위**: 고객사별, 카테고리별, 담당자별 지연 패턴 분석
- **목적**: 시간 관리 개선을 위한 구체적 전략 제시

#### 2.3.2 지연 패턴 분석 로직
```javascript
function analyzeDelayPatterns(cars) {
  const continuousEvents = cars.filter(car => car.eventType === 'CONTINUOUS');
  
  const delayAnalysis = {
    totalEvents: continuousEvents.length,
    onTimeEvents: continuousEvents.filter(car => car.dateScore >= 3).length,
    delayedEvents: continuousEvents.filter(car => car.dateScore < 0).length,
    avgDateScore: calculateAverage(continuousEvents.map(car => car.dateScore)),
    categoryDelays: analyzeCategoryDelays(continuousEvents),
    improvementAreas: identifyImprovementAreas(continuousEvents)
  };
  
  return delayAnalysis;
}
```

#### 2.3.3 시간 관리 효율성 지표
- **기한 준수율**: (기한 내 완료 건수 / 전체 CONTINUOUS 건수) * 100
- **평균 지연 일수**: 지연된 이벤트의 평균 지연 일수
- **지연 빈도**: 카테고리별 지연 발생 빈도
- **개선 가능성**: Date Score 향상 여지 정량화

```javascript
function generateDateScoreEvidence(cars) {
  const continuousEvents = cars.filter(car => car.eventType === 'CONTINUOUS');
  
  if (continuousEvents.length === 0) {
    return "시간 관리 분석: CONTINUOUS 이벤트 없음";
  }
  
  const onTimeEvents = continuousEvents.filter(car => car.dateScore >= 3).length;
  const delayedEvents = continuousEvents.filter(car => car.dateScore < 0).length;

  return `시간 관리 효율성: 기한준수 ${onTimeEvents}건(${onTimeRate}%), 지연 ${delayedEvents}건. ` +
         `개선 집중 영역: 사전 점검 프로세스 강화 필요.`;
}
```

```javascript
async function generateAIAnalysis(customer) {
```

```javascript
function validateScoreConsistency(cars) {
  const inconsistencies = [];
  
  cars.forEach(car => {
    // Date Score 정합성 검증
    if (car.eventType === 'CONTINUOUS' && car.dueDate && car.completionDate) {
      const calculatedDateScore = calculateDateScore(car.dueDate, car.completionDate);
      if (Math.abs(car.dateScore - calculatedDateScore) > 0.1) {
        inconsistencies.push({
          carId: car.id,
          type: 'date_score_mismatch',
          expected: calculatedDateScore,
          actual: car.dateScore
        });
      }
    }
  });
  
  return inconsistencies;
}
```

```javascript
function generateRiskEvidence(cars) {
```
